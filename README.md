# Bucket-player

**Bucket-player** is a platform currently under development that allows users to fetch and stream videos directly from an Amazon S3 bucket, offering a YouTube-like experience.

## Cores Technologies

- **Amazon S3 Bucket API**: for storing and video content
- **React + Vite**: Front-end
- **Tailwind CSS + Material Tailwind**: user interface
